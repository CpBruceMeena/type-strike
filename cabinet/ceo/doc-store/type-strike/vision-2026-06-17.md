## Problem Statement

### Target User
Mobile gamers aged 18–35 who crave competitive, fast-paced arcade experiences and want to sharpen their typing fluency on mobile — without feeling like they're doing homework.

### The Problem
Typing "games" today are flat, web-first utilities that feel like productivity tools dressed in game clothing. They lack:
- Mobile-native design (tiny keys, no thumb-optimized layouts)
- Visual excitement (flat design, no feedback, no stakes)
- Progression systems (no sense of journey or mastery)
- Competitive depth (no real combos, no risk/reward)

### Why Now
- Mobile gaming is the largest gaming segment globally ($100B+)
- Typing on phones is ubiquitous — everyone types every day
- Existing typing games (Monkeytype, TypeRacer) haven't evolved in a decade
- There's a gap for an ultra-premium, visually stunning mobile typing game that treats typing as a **skill to master**, not a chore to measure

### Current Alternative
- **Monkeytype** — web-only, flat design, no mobile optimization, no game mechanics
- **TypeRacer** — dated UI, web-first, car-racing theme doesn't translate to mobile
- **NitroType** — Flash-era aesthetics, minimal mobile support
- **Generic word games** (Wordle, Wordscapes) — not typing-based, no speed component

### Core Value Proposition
> **"A hyper-stylized competitive typing game that turns every word into an explosive arcade battle."**

## Product Vision

### Name & Tagline
**type-strike** — *"Type with fury. Strike with fire."*

### Elevator Pitch
type-strike is a mobile competitive typing game that abandons flat, boring typing tests for an explosive 3D arcade experience. Players journey through a broken-tech progression map of 100+ levels, type words that shatter like crystalline glass on correct input, and ride a liquid-fueled plasma combo engine to glory. Every keystroke is a weapon. Every mistake cracks the glass. Every streak ignites the UI.

### Key Differentiators
1. **Hyper-stylized 3D visual identity** — obsidian/slate chassis, magma/gold/neon accents, smoked glass panels, 2.5D perspective with chiseled bevels
2. **Violent particle feedback** — letters shatter like glass shards with neon trails on correct input; red electrical glass cracks on incorrect input
3. **Liquid-fueled combo engine** — "Overdrive Gauge" fills with raging plasma fire fluid as streaks grow; UI pulses with motion blur; razor-sharp kinetic text: "CRITICAL COMBO!", "MAX FRENZY!", "IGNITION SPEED!"
4. **Mechanical keycaps on mobile** — aggressively angled keycaps with physical depth, metallic side moldings, premium tactility
5. **100+ crystalline node progression** — floating 3D keycap level nodes with plasma fire cores (unlocked) or metallic charcoal shields (locked); golden crystal shard trophies

### Target Market
- **B2C** — Mobile gamers, competitive typists, word game enthusiasts
- **Primary:** 18–35 year old mobile gamers
- **Secondary:** Typists looking for fun practice (students, writers, devs)
- **Psychographic:** Craves mastery, visual spectacle, competitive validation

### Geographic Scope
- **Global, English-first** — launch in English-speaking markets (US, UK, Canada, Australia, India)
- **Expansion:** Localized word packs for top 10 languages

## Phased Feature Roadmap

### Phase 1: MVP — *The Core Blast* (Estimated: Medium)
**Goal:** Ship the core gameplay loop that validates the concept and visual identity.

| Screen | Features |
|--------|----------|
| **Map Screen** | Vertical broken-tech path with 25 levels; floating crystalline keycap nodes (unlocked = plasma fire core, locked = metallic charcoal shield); golden crystal shard trophies |
| **Gameplay Arena** | Upper word display panel (smoked thick glass, letters emerge from volumetric glowing void); custom thumb-typing zone with mechanical keycaps; Overdrive Combo Gauge (liquid plasma fuel chamber) |
| **Victory Assessment** | Full-screen overlay with raw metal brackets; "VICTORY ATTAINED" header badge with digital smoke particles; WPM & accuracy in hyper-stylized geometric typography; curved XP arc bar that aggressively fills; three floating golden shard trophies |
| **Core Tech** | Touch input detection, word bank (500+ words), combo/score system, session persistence |

| Feature | Details |

| Account System | Anonymous + full account, cloud save across devices |

### Phase 3: Scale — *Multiplayer & Monetization* (Estimated: Large)
**Goal:** Build social layer, unlock revenue, drive viral growth.

| Feature | Details |

**Goal:** Adjacent markets, user-generated content, esports.

| Feature | Details |

| Multi-Language | Word packs for top 10 non-English languages (Spanish, Mandarin, Hindi, Arabic, French, German, Japanese, Korean, Portuguese, Russian) |
| Custom Level Creator | Players design and share their own typing challenges |
| Spectator Mode | Watch live multiplayer matches with commentary overlay |

## Competitive Landscape

### Direct Competitors

| Competitor | Strengths | Weaknesses |
|------------|-----------|------------|
| **Monkeytype** | Huge web audience, extensive stats, open-source | No mobile app, flat design, no game mechanics |
| **TypeRacer** | Established brand, multiplayer racing | Dated UI, web-only, car theme doesn't translate to mobile |
| **NitroType** | Fun car-typing hybrid, school adoption | Flash-era design, minimal mobile UX, declining user base |
| **Keybr** | Adaptive learning algorithm, clean design | Boring presentation, no competitive element, web-only |
| **Typing.com** | Educational market, teacher dashboard | Utilitarian, not fun, no game mechanics |

### Indirect Competitors
- **Wordle** — daily word puzzle (different mechanic, same brain space)
- **Wordscapes** — casual word game (relaxed, not competitive)
- **Clash Royale** — competitive mobile game (different genre, same audience)
- **Brawl Stars** — fast-paced mobile PvP (same dopamine loop)

### Our Moat
1. **Visual Identity** — hyper-stylized 3D arcade presentation is unmatched in the typing game space
2. **Mobile-Native Design** — built for thumbs, not keyboards; every competitor is web-first
3. **Plasma Combo Engine** — the core risk/reward mechanic (streak multiplier decay) is our unique game loop
4. **Progression Depth** — 100+ levels with visual payoff creates meaningful mastery journey
5. **Feedback Fidelity** — violent particle shatter, glass cracking, liquid fuel gauge — the tactile/visual loop keeps players coming back

### Market Positioning
> **"The Call of Duty of typing games"** — premium, aggressive, visually stunning, competitively deep. We don't compete with typing *utilities*; we compete with mobile *arcade games* for attention.

## Risks & Mitigations

| Risk | Probability | Impact | Mitigation |
|------|-------------|--------|------------|
| **Typing games are a niche audience** | Medium | High | Position as an arcade game first, typing game second — compete in mobile gaming, not typing utilities |
| **High-fidelity 3D visuals strain mobile performance** | Medium | Medium | Early device profiling; target mid-range phones (iPhone 11+/Android equivalent); LOD system for older devices |
| **Touch typing accuracy issues** | High | High | Custom thumb keyboard zone with aggressive hitbox tuning; calibration tool; key size adjustment settings |
| **Content exhaustion (500 words get old)** | Medium | Medium | Daily challenges, word packs as DLC, user-generated content in Phase 4 |
| **Monetization resistance** | Medium | Medium | Free tier must feel generous; cosmetics only, no pay-to-win; generous trial period for premium |
| **Competitive cloning** | Low | High | Moat is visual identity + mobile-native design; trademark name and key visual elements; patent combo gauge mechanic if novel |

## Tech Stack Recommendations

| Layer | Recommendation |
|-------|---------------|
| **Game Engine** | Unity (best mobile 3D performance + particle systems + mobile deployment) or Godot (open-source, lighter) |
| **UI Framework** | Unity UI Toolkit + custom shaders for glass/plasma effects |
| **Backend** | Firebase (auth, cloud save, leaderboards, analytics) → scale to custom backend in Phase 3 |
| **Analytics** | Firebase Analytics + Mixpanel (deep event tracking) |
| **Crash Reporting** | Sentry or Crashlytics |
| **CI/CD** | GitHub Actions → TestFlight (iOS) + Play Console (Android) |
| **Ads** | AdMob (rewarded + interstitial) |
