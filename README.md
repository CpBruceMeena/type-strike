# ⚡ type-strike

**Type with fury. Strike with fire.**

type-strike is a mobile typing game that turns every word into an explosive arcade battle. No flat typing tests — just visceral feedback, liquid-fueled combos, and a journey through 100+ levels of fire and fury.

---

## The Core Loop

| Step | What Happens |
|------|-------------|
| **1. Launch** | You enter a dark, molten world — your progression map |
| **2. Pick a level** | Each node is a crystalline keycap waiting to be conquered |
| **3. Type** | Words appear. You type them. Correct keystrokes shatter the letters. Mistakes crack the glass. |
| **4. Build the combo** | Streaks fill a raging plasma gauge. The longer the streak, the more the UI ignites. |
| **5. Pass or fail** | Meet the WPM and accuracy targets to advance. Fall short, and the level is breached. |
| **6. Rise** | Earn XP, level up, collect stars, and unlock the next tier of challenges. |

---

## Features

- **100+ levels** across 4 ascending tiers: Ember → Igneous → Magma Core → Obsidian
- **Plasma combo engine** — build streaks to fuel a liquid overdrive gauge
- **Custom thumb keyboard** — mechanical-style keycaps designed for mobile typing
- **Kinetic feedback** — shattering glass, crack effects, particle storms
- **Rich progression** — XP, levels, titles, stars, and a sprawling broken-tech map
- **Configurable settings** — keyboard layout, key sounds, haptics, visual effects, accessibility
- **Dark arcade aesthetic** — obsidian, magma red, molten gold, and neon purple
