# ⚡ Type Strike

<p align="center">
  <strong>Type with fury. Strike with fire.</strong>

A typing game that turns every word into an explosive arcade battle. No flat typing tests — just visceral feedback, combo-fueled streaks, and a journey through fire and fury.

---

## 🔥 What is Type Strike?

Type Strike transforms typing practice into a gamified, arcade-style experience. Every correct keystroke builds combo energy. Every mistake costs momentum. Progress through 500 levels of escalating difficulty across 5 elemental tiers — from Ember to Beyond.

---

## 🎮 Game Modes

| Mode | Description |
|------|-------------|
| **Strike (Levels)** | 500 levels across 5 tiers — Ember, Igneous, Magma Core, Obsidian, Beyond. Clear paragraphs with speed and precision to earn up to 3 stars per level. |
| **Timed Sprints** | Test your raw speed in 1-minute, 3-minute, or 5-minute bursts. Compete on the timed leaderboard. |
| **Coder Mode** | Type real multi-line code snippets across 7 programming languages (JavaScript, TypeScript, Python, Go, Java, C++, Rust). Covers algorithms, data structures, and DSA patterns. |
| **Learn Mode** | 48 progressive typing lessons covering fundamentals to advanced techniques. |
| **Daily Contests** | One paragraph, one attempt, ranked globally. Return every day for a new challenge. |

---

## 🏆 Features

- **Combo Engine** — 6 escalating tiers: Igniting → Burning → Critical Combo → Max Frenzy → Ignition Speed. Visual and audio feedback at every tier.
- **Leaderboards** — Global (XP), Daily (today's best), and Timed (1min/3min/5min) rankings. See where you stand against all players.
- **Progression System** — 7 rank tiers (Recruit → Bronze → Silver → Gold → Platinum → Diamond → Obsidian). Unlock titles and keyboard themes as you climb.
- **Streak Rewards** — 30-day reward calendar with bonus XP milestones at 5, 10, 15, 20, 25, and 30 days. Freeze tokens protect your streak.
- **Achievements** — 30+ feats across speed, accuracy, combo, progression, and streak categories.
- **Stats Dashboard** — Track your game history, level progress, accuracy trends, and per-mode performance.
- **Share Cards** — Dynamic Open Graph images for social media sharing of victories and achievements.
