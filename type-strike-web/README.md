# 🔥 Type Strike — Web

<p align="center">
  <strong>Type with fury. Strike with fire.</strong>
</p>

The browser-based version of Type Strike — a typing game that turns every word into an explosive arcade battle. Play timed sprints, daily contests, code typing challenges, and progress through 500 levels of fire and fury.

---

## 🎮 Game Modes

| Mode | Duration | Description |
|------|----------|-------------|
| **1 Min Sprint** | 60s | Quick-fire speed test |
| **3 Min Endurance** | 180s | Pace yourself through a longer paragraph |
| **5 Min Marathon** | 300s | Full stamina challenge |
| **Daily Contest** | — | One attempt, one expert paragraph, ranked globally |
| **Coder Mode** | Unlimited | Type real code snippets — algorithms, data structures, and DSA patterns across 7 languages |

---

## 🏆 Leaderboards

Compete across three leaderboard categories:
- **Global** — Ranked by total XP across all game modes
- **Daily** — Best performance in today's daily challenge
- **Timed** — Fastest WPM in 1-minute, 3-minute, and 5-minute sprints

---

## 🔐 Authentication

Sign in with **email**, **Google**, or **GitHub** to persist your progress, track achievements, and appear on leaderboards. No account required to try timed modes.
